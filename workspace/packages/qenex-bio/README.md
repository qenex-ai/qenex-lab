# @qenex/bio

This package is the domain expert for Biological Systems and Genomics within the QENEX LAB.

## Capabilities

- **Protein Folding**: Interfaces for AlphaFold/OpenMM for structural biology prediction.
- **Genomics**: Sequence alignment, variant calling, and CRISPR off-target analysis.
- **Systems Biology**: Pathway modeling and metabolic flux analysis.
- **Evolutionary Dynamics**: Simulation of population genetics and selection pressures.

## Integration

Solves "Silent Failure" modes by binding to real physical simulation engines instead of relying on LLM token prediction for biological processes.

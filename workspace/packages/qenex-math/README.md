# @qenex/math

This package is the domain expert for Formal Mathematics and Proof Verification within the QENEX LAB.

## Capabilities

- **Formal Verification**: Bridges to Lean 4 / Coq for automated theorem proving.
- **Symbolic Computation**: Computer algebra systems for exact solution derivation.
- **Numerical Analysis**: High-precision arbitrary-precision arithmetic.
- **Complexity Analysis**: Automated worst-case execution time and space complexity derivation.

## Strategy

Addresses "Infinite Recursion" failures by outsourcing proof step verification to formal logic engines rather than relying on probabilistic LLM reasoning.

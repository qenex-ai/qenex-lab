# @qenex/chem

This package focuses on quantum chemistry and molecular dynamics simulations.

## Capabilities

- **Quantum Solvers**: Interfaces for Hartree-Fock, DFT, and Coupled Cluster methods (Psi4/PySCF).
- **Molecular Design**: Tools for structure generation and optimization.
- **Reaction Pathways**: Transition state finding and barrier calculation.

## Integration

Provides the backend for chemical reasoning tasks in the Trinity Pipeline.

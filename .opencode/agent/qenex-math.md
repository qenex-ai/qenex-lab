## Proof Standards

When writing proofs:

1. State all assumptions clearly
2. Define all terms and notation
3. Use rigorous logical steps
4. Consider edge cases and counterexamples
5. Verify with computational checks when possible

## Numerical Best Practices

- Use appropriate precision for the problem
- Check condition numbers for linear systems
- Monitor convergence of iterative methods
- Validate against analytical solutions when available
